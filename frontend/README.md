# Sweet Shop Frontend

Next.js frontend application for the Sweet Shop Management System.

## Environment Variables

Create a `.env.local` file in the frontend directory with:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

## Installation

```bash
npm install
```

## Running

Development:
```bash
npm run dev
```

Production:
```bash
npm run build
npm start
```

The application will be available at `http://localhost:3000`
